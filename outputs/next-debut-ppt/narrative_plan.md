# NEXT DEBUT 발표 PPT 내러티브 플랜

## Audience
- 학교 프로젝트 발표 청중
- 서비스 흐름과 구현 범위를 빠르게 이해해야 하는 평가자

## Objective
- NEXT DEBUT가 단순 게임이 아니라 K-POP 연습생 성장 과정에 몰입하는 팬 참여형 플랫폼임을 전달한다.
- 게임, 커뮤니티, 상점, AI, 실시간 채팅, 관리자 기능까지 통합 구현한 점을 보여준다.

## Narrative Arc
1. 서비스 첫 인상과 세계관 제시
2. 왜 이 서비스를 만들었는지 설명
3. 어떤 기능과 구조로 구현했는지 실제 화면 중심으로 소개
4. 팀 협업과 개발 프로세스를 정리
5. 확장 가능한 플랫폼이라는 메시지로 마무리

## Visual System
- 메인 컬러: 핑크, 라벤더, 화이트, 크림
- 포인트 컬러: 민트, 골드, 소프트 블루
- 밝은 톤 배경 위에 유리판 카드와 부드러운 그라데이션 사용
- 진한 회색 텍스트로 가독성 확보
- 실제 프로젝트 UI와 닮은 서비스 소개형 레이아웃 유지

## Source Plan
- 사용자가 제공한 실제 서비스 스크린샷 사용
- 구조도와 ERD도 제공된 파일을 활용
- 일정, 역할, 개발환경, 마무리 슬라이드는 네이티브 텍스트와 카드 레이아웃으로 구성

## Editability Plan
- 제목, 본문, 표, 강조 문구는 모두 편집 가능한 텍스트 객체로 제작
- 스크린샷은 이미지 프레임으로 삽입
- 일정과 역할 슬라이드는 편집 가능한 카드형 텍스트와 표로 제작
